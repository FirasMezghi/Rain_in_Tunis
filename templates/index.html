About Rainy in Tunis

Rainy Tunis is a web app which has a Machine Learning model running at the back. The purpose of developing this app is to predict whether it will rain the next day or not in Tunisia. This model is based on dataset that has been web scarped from the website tuiempo. The codes for this project can be checked in my github repo.
